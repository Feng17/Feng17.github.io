JDBC

发表于 2017-02-11

JDBC(java database connectivity)为Java数据库连接，是一种用于执行SQL语句的Java API，JDBC就是用Java语言来向各种关系型数据库发送sql语句从而来操作数据库。这次通过简单的学生信息管理的例子来总结下JDBC的一些知识和SQL的一些基本语句。

使用JDBC代码查询数据库的信息流程是我们需要创建一个数据库连接，然后再创建一个语句对象，接着进行查询；在最后我们需要关闭数据库连接、语句和结果集(关闭资源时采用倒关的手法)。看上去挺麻烦的，还需要捕获SQLException，不过最近看到其他书上写的之后学习了Spring后可以使用Spring的JdbcTemplate来进行操作避免JDBC的样板式代码，这个在之后框架再好好学习下。

SQL数据中的属性类型

  • int:4字节，整型,大整数值。tinyint:1字节,小整数值。 smallint:2字节，大整数值。 也可以加上unsigned关键字，定义成无符号的类型。

  • float(m, d):4字节，单精度浮点型，m总个数，d小数位。 double(m, d):8字节，双精度浮点型，m总个数，d小数位。decimal(m, d):decimal是存储为字符串的浮点数。

  • char(n):字符型，固定长度字符串类型char(255); 数据的长度不足指定长度，它会补足到指定长度。

    varchar(n):可变长度字符串类型varchar(65535),存入的数据多长就是多长。varchar会使用1-3个字节来存储长度，text不会。括号中n代表字符的个数，并不代表字节个数。

    text:MySQL独有的数据类型text(65535)，字符串类型。

  • date:3字节，日期类型，格式：YYYY-MM-DD

    time:3字节，时间类型，格式为:hh:mm:ss

    datetime:8字节，日期时间，格式：:YYYY-MM-DD hh:mm:ss

  • blob:blob字节类型常常是数据库中用来存储二进制文件的字段类型,典型的blob是一张图片或一个声音文件。

DML语法

  • 插入数据: insert into 表名 (列名1，列名2，列名3) values (列值1，列值2，列值3) 一次插入就是一行

  • 修改数据: 修改某列的全部值:update 表名 set 列名称 =新值

    修改列的指定值:update表名称 set 列名称 = 新值 where列名称 = 某值

  • 获取数据: select 列名称 from 表名称 select* from 表名称 –星号（*）是选取所有列的快捷方式

    如需有条件地从表中选取数据，可将 where子句添加到 select 语句

    例如 select * from student where age='20'

  • 删除数据:delete语句用于删除表中的行 delete from 表名称 where 列名称 = 值

JDBC对数据库进行CRUD操作

PreparedStatement

PrepaerdStatement是Statement的子接口，Statement接口用于执行静态SQL语句并返回它所生成结果的对象。PrepaerdStatement对象比Statement对象效率更高，而且也可以防止SQL注入，所以我们一般用PrepaerdStatement。

常用Statement方法：

1.excute():运行语句，返回是否有结果集。

2.excuteQuery():运行select语句，返回ResultSet结果集。

3.excuteUpdate():运行insert/update/delete语句，返回更新的行数。

ResultSet

Statement执行SQL语句时返回ResultSet结果集。

常用ResultSet方法：

1.getInt():获得在数据库里时int类型的对象。

2.getString():获得在数据库里时varchar,char类型的对象。

3.getFloat():获得在数据库里时float类型的对象。

4.getDate():获得在数据库里时date类型的对象。

5.getBoolean():获得在数据库里时boolean类型的对象。
